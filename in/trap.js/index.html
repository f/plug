How to use?

Has a very easy usage. There is no option to put. 

$('selector').trap();

Requirements

When using trap, you require adding these CSS classes into your project: 

.trapped {position: fixed !important; top: 0 !important;}
.trap-after {visibility: hidden; margin-top: 0; height: 0;}
